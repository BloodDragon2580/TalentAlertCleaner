# English
# TalentAlertCleaner

No more red dots. No more pulsing. No more UI clutter.

**TalentAlertCleaner** automatically removes notification dots (“alert indicators”) and microbutton pulsing from selected MicroButtons – especially the Talent/Spellbook button.

This addon is designed specifically for **WoW Retail / Midnight** and uses only secure `hooksecurefunc` hooks. It does not modify protected frames directly and causes **no taint or forbidden action errors**.

---

## ✨ Features

- Removes Talent/Spellbook notification dot
- Stops microbutton pulsing
- Fully Retail / Midnight compatible
- No modification of protected Blizzard code
- No performance overhead

---

## 🛡 Clean & Safe

TalentAlertCleaner:
- does not override Blizzard functions
- does not manipulate protected frames directly
- uses secure hooks only
- is taint-safe

---

## 🎯 Who is this addon for?

For players who:
- prefer a clean UI without constant notification dots
- don’t want repeated talent reminders
- dislike microbutton pulsing
- enjoy a minimal interface

Install and forget – it works automatically with no configuration required.

Lightweight. Stable. Invisible.
